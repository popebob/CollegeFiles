    • For users of  Internet Explorer only:

      When you save a page already  loaded into Internet Explorer's  window,  it creates a folder on the client computer to hold all images located on that page.  (Using Explorer, therefore you will have two copies of the cheeta.gif file saved to your hard drive.)   Explorer also, modifies the page that is downloaded, changing directory names used to reference images.  

      View the source file for this page  and you will see this line following the title line:
      <img src="images/cheeta.gif" width="140" height="60" alt="cheeta.gif (7232 bytes)">

      Viewing the page after downloading you will see:

      <img src="catPage_files/cheeta.gif" width="140" height="60" alt="cheeta.gif (7232 bytes)">

      Where the text in red was inserted by Explorer.  Change catPage_files back to images
